🧠 AI News Auto-Blogger & Email Automation

 🚀 Overview
This project is an automated AI-powered workflow that fetches the latest AI news, generates blog articles using an LLM, and sends them via email.

 ⚙️ Tech Stack
- n8n (workflow automation)
- SerpAPI (news fetching)
- Groq LLM (blog generation)
- Gmail API (email delivery)

 🔄 Workflow Architecture
Trigger → Fetch News → Filter → LLM → Format → Email

 ✨ Features
- Fully automated daily execution
- Real-time AI news fetching
- AI-generated blog content
- Email delivery system

 📂 Files
- workflow.json → n8n workflow export

 🧠 How it Works
1. Scheduled trigger runs daily
2. Fetches latest AI news using SerpAPI
3. Filters top articles
4. Generates blog using Groq LLM
5. Sends formatted blog via email

 🚀 Future Improvements
- Multi-article summarization
- Blog publishing (Medium/WordPress)
- Dashboard UI
